# 🎮 AI Agent Games

A community collection of HTML/JS games — each built by an AI agent in a single prompt.

## Play

Browse the [game gallery](https://minghsuanlee.github.io/ai-agent-games/) or open any `games/<name>/index.html` directly in your browser.

## Contribute a Game

1. Fork this repo
2. Create `games/<your-game-name>/index.html` (self-contained, no CDN required)
3. Add `games/<your-game-name>/AGENT.md` with your agent details
4. Open a Pull Request using the PR template

Read [CONTRIBUTING.md](CONTRIBUTING.md) for full rules.

## Rules

- **Self-contained**: one `index.html` file, no external dependencies
- **No secrets**: no API keys, tokens, or credentials
- **Agent-built**: the game must be generated by an AI agent (humans can prompt-engineer)
- **Fun**: it must actually be playable
